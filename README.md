# Syntexchub_Hand_gesture_Recognition

✋ Hand Gesture Recognition System
🚀 A real-time hand gesture recognition system built using OpenCV and MediaPipe.
This project detects hand landmarks via webcam and classifies gestures to simulate actions like play, pause, volume control, and more.

---

📌 Overview

This project demonstrates how computer vision + hand tracking can be used to create intuitive human-computer interaction systems.

---

✨ Features

- 🎥 Real-time webcam-based hand tracking
- 🖐️ 21 hand landmark detection using MediaPipe
- 🎯 Gesture classification system
- ⚙️ Gesture-to-action mapping
- ⚡ FPS counter for performance tracking

---

🖐️ Supported Gestures

Gesture| Action
✊ Fist| Pause ⏸
✋ Open Palm| Play ▶
👍 Thumb Up| Like 👍
☝ Index Finger| Volume Up 🔊
👇 Pinky Finger| Volume Down 🔉
✌ Peace| Next Track ▶▶
🤟 Rock Sign| Mute 🔇

🧠 How It Works

1. Captures video from webcam
2. Detects hand landmarks using MediaPipe
3. Analyzes finger positions
4. Classifies gesture
5. Displays corresponding action

---

📸 Output Preview

- Live webcam feed
- Hand landmarks drawn
- Gesture + Action text displayed
- FPS counter

---

⚠️ Limitations

- Requires good lighting for accuracy
- Works best with right hand (thumb logic)
- Actions are simulated (no real system control yet)
